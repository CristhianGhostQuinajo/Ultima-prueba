Windows Pro

Windows es el sistema operativo insignia de Microsoft, el estándar de facto para las computadoras domésticas y de negocios. El sistema operativo basado en una interfaz gráfica de usuario (GUI) fue introducido en 1985 y se han liberado muchas versiones desde entonces, como se describe a continuación y con un precio que esta en oferta es tan solo a 8.990 CLP con un IVA del 19%.

Caracteristicas de Windows Pro
Interfaz familiar para una transición fácil.
Seguridad avanzada con Windows Defender y actualizaciones regulares.
Productividad máxima con Cortana y escritorios virtuales.
Compatibilidad profesional para aplicaciones esenciales.
Gestión simplificada de dispositivos para administradores.
